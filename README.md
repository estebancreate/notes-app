# Notes App

## 项目简介
一个支持 Markdown 的笔记应用，用户可以创建、编辑和删除笔记，并保存到本地。

## 功能
- 创建、编辑、删除笔记
- Markdown 语法实时渲染（Marked.js）
- 本地存储笔记

## 技术栈
- HTML, CSS, JavaScript, Marked.js, localStorage
- 部署：Vercel
